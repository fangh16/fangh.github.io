# Huang Fang Personal Website

This is a static academic website designed for GitHub Pages.

## Pages

- `index.html`: homepage and overview
- `resume.html`: CV, research interests, education, skills, and talks
- `publications.html`: selected publications with representative images and summaries

## Deploy on GitHub Pages

1. Create a GitHub repository, for example `huangfang.github.io` or `personal-website`.
2. Upload all files in this folder to the repository.
3. In GitHub, open `Settings` → `Pages`.
4. Choose `Deploy from a branch`, then select the `main` branch and the root folder.
5. Save. GitHub will publish the site after a short build.

If the repository is named `huangfang.github.io`, the site address will be `https://huangfang.github.io/`.
For any other repository name, it will usually be `https://USERNAME.github.io/REPOSITORY/`.
